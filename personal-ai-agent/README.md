# Личный ИИ-агент — Аналитик идей

Это мой собственный ИИ-агент на Python, который помогает:

- Хранить идеи навсегда (сохраняются в файл)
- Добавлять идеи (многострочный ввод)
- Просматривать список (коротко и полностью)
- Искать по ключевому слову
- Редактировать любую идею
- Показывать статистику
- Экспортировать всё в Markdown-файл
